# Canal Crush 

A Dutch-themed match-three puzzle game built with React. Swap and match iconic Dutch symbols — tulips, stroopwafels, clogs, windmills, bikes, and cheese — across eight levels set in cities from Delft to Rotterdam.

(vibe-coded)

## Features

- 7×8 match-three grid with swipe/click controls
- 8 progressively harder levels, each with a Dutch city theme and unique collection targets
- "Tide" mechanic that raises the water level as levels progress
- Procedural sound effects via [Tone.js](https://tonejs.github.io/)
- Mobile-first, touch-friendly layout

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
```

Output is written to `dist/`.

## Tech Stack

- [React 18](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [Tone.js](https://tonejs.github.io/)
